PaceKeeper Help

What is PaceKeeper?

PaceKeeper is a free curling game monitor. Mount it behind the glass to keep club games on schedule, and ensure everyone knows the rule when the buzzer sounds.

Version 1.0.1

How to Use

1. Configure the Match — Use the arrows to set the hours and minutes for your game duration.

2. Set the Policy — Lock in the club's rule for when the timer hits zero (e.g., finish the current end, or play one more).

3. Start the Clock — Tap 'Set Timer', then hit the large Start button when the first rock is thrown.

4. Manage Time — Pause for timeouts or injuries. Use the left reset button to restart the clock, or the right home button to return to setup.

5. Alarm Dismissal — When the timer hits zero, the main button turns into a large "DONE" button. Clicking this immediately silences the buzzer and hides the visual animation.

Display Modes

Toggle between Digital and Analog views using the icons in the top right. The Analog view acts as a dynamic visual pie-chart, making it easy for skips to estimate remaining time from across the sheet.

PaceKeeper has full support for both portrait and landscape orientations.

Keep the Screen On

PaceKeeper uses modern browser features (Wake Lock API) to explicitly request that your device does not turn off the screen while a match is in progress.
